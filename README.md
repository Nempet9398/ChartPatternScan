# ChartPatternScan

종목과 기간을 선택하면, 현재 차트가 어떤 고전 패턴과 가장 닮았는지 유사도(%)로 알려주는 무료 웹 서비스입니다.

---

## 어떤 서비스인가요?

차트를 보다가 "이거 헤드앤숄더 같은데?" 싶을 때, 직접 수치로 확인할 수 있습니다.
AI나 딥러닝 없이, 수학적 규칙만으로 패턴을 분석합니다.

---

## 주요 기능

### 종목 검색
- 한국 주식 (코스피 / 코스닥), 미국 주식, 암호화폐 모두 지원
- 종목명 또는 코드로 검색 가능
- 예: 삼성전자, AAPL, BTC

### 기간 선택
- 1주 / 1개월 / 3개월 / 6개월 / 1년 중 선택

### 캔들스틱 차트
- 선택한 종목과 기간의 OHLCV 차트 표시
- 패턴이 감지된 구간은 차트 위에 하이라이트로 표시

### 패턴 분석 — Top 3
현재 차트와 가장 유사한 패턴 3개를 유사도 순으로 보여줍니다.

각 패턴 카드에는 이런 정보가 담깁니다:
- 패턴 이름 (한글 + 영문)
- 유사도 % (컬러 바로 시각화)
- 신호 방향: 상승 / 하락 / 중립
- 역사적 성공률 (실제 통계 기반)
- 패턴 설명 한 줄

## 감지 가능한 패턴 (총 18개)

| 패턴 | 신호 | 설명 |
|------|------|------|
| 헤드앤숄더 | 하락 | 세 고점 중 가운데가 가장 높은 하락 반전 |
| 역헤드앤숄더 | 상승 | 헤드앤숄더 반전, 상승 신호 |
| 더블탑 | 하락 | 비슷한 두 고점 후 하락 (M자형) |
| 더블바텀 | 상승 | 비슷한 두 저점 후 상승 (W자형) |
| 트리플탑 | 하락 | 세 번 저항받고 하락 반전 |
| 트리플바텀 | 상승 | 세 번 지지받고 상승 반전 |
| 골든크로스 | 상승 | 단기 이동평균이 장기 이동평균 상향 돌파 |
| 데드크로스 | 하락 | 단기 이동평균이 장기 이동평균 하향 이탈 |
| 대칭 삼각수렴 | 중립 | 고점 하향 + 저점 상향으로 수렴 |
| 상승 삼각수렴 | 상승 | 저항선 수평 + 저점 상향 수렴 |
| 하락 삼각수렴 | 하락 | 지지선 수평 + 고점 하향 수렴 |
| 직사각형 | 중립→방향 | 박스권 횡보 후 돌파 방향 추적 |
| 상승 쐐기 | 하락 | 둘 다 우상향이나 점점 수렴, 하락 반전 |
| 하락 쐐기 | 상승 | 둘 다 우하향이나 점점 수렴, 상승 반전 |
| 불 깃발 | 상승 | 급등 후 평행 채널 조정, 상승 지속 |
| 베어 깃발 | 하락 | 급락 후 평행 채널 반등, 하락 지속 |
| 불 페넌트 | 상승 | 급등 후 수렴 삼각형 조정, 상승 지속 |
| 베어 페넌트 | 하락 | 급락 후 수렴 삼각형 반등, 하락 지속 |
